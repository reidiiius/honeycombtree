### Usage
Command-line

    cd honeycombtree/

:

    cargo build

:

    cargo run

:

    cargo run tonal

:

    cargo run group wq xq

:

    cargo run n0 k1

:

    cargo run n0 k1 query

:

    cargo run n0yy k1zz

:

    cargo run cgdae n0yy k1zz

:

    cargo run query k2j j2k

:

    cargo run beadgcf k2j6 j2k5

:

    cargo run beadgcf gamut | sensible-pager
