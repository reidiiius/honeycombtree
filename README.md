### Usage
Command-line

    cd honeycombtree/

:

    cargo build

:

    cargo run

:

    cargo run tonal

:

    cargo run group wq xq

:

    cargo run n0 j3

:

    cargo run cgdae n0 j3

:

    cargo run query k2j j2k

:

    cargo run gamut | sensible-pager
